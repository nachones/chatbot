Ejemplo de Chatbot Inteligente

Esta página demuestra cómo integrar el chatbot inteligente en tu sitio web.

Características del chatbot:

  • ✅ Chat en tiempo real
  • ✅ Diseño responsive
  • ✅ Entrenamiento personalizado
  • ✅ Soporte para múltiples tipos de archivos
  • ✅ Integración sencilla

Código de integración:

<!-- Agrega este código justo antes del </body> -->
<script src="https://app.micopiloto.es/chat-widget.js"
  data-api-url="https://app.micopiloto.es/api"
  data-api-key="TU_CHATBOT_ID"
  data-title="Asistente Virtual"
  data-welcome="¡Hola! ¿En qué puedo ayudarte hoy?"
  data-primary-color="#6366F1">
</script>

Personalización:

Puedes personalizar el chatbot con estos parámetros:

  • data-api-key: ID del chatbot (obtenido del dashboard)
  • data-api-url: URL del servidor API
  • data-title: Título del chat
  • data-welcome: Mensaje de bienvenida
  • data-primary-color: Color principal (hex)
  • data-position: bottom-right, bottom-left

Instrucciones de uso:

  1. Asegúrate de que el servidor esté ejecutándose en https://app.micopiloto.es
  2. Configura tu API key de OpenAI en el archivo .env
  3. Entrena el modelo con tus documentos usando los endpoints de entrenamiento
  4. Copia el código de integración arriba en tu sitio web
